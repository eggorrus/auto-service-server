# Сервер для приложения "Управление автосервисом"

Сервер на чистой Java, обрабатывающий запросы от клиентского приложения и взаимодействующий с PostgreSQL через JDBC.

## Технологии
- Java 17
- PostgreSQL
- JDBC
- Maven

## Запуск
1. Настройте подключение к БД в `server-config.properties`
2. Запустите 'ServerApp'
